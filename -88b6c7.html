三个线程交替打印（ABC或者数字）（wait，notify，syn方法）

public class Demo {
    private static int num=10;
    private static int state=0;
    private static Object object=new Object();
    public static void fun(String name,int sign){
        for(int i=0;i<num;i++){
            synchronized (object){
                while(sign%3!=sign){
                    try {
                        object.wait();
                    } catch (InterruptedException e) {
                        throw new RuntimeException(e);
                    }
                }
                System.out.println(state);
                state++;
                object.notifyAll();
            }
        }
    }

    public static void main(String[] args) {
        new Thread(()->{
            Demo.fun("A",0);
        }).start();
        new Thread(()->{
            Demo.fun("A",0);
        }).start();
        new Thread(()->{
            Demo.fun("A",0);
        }).start();
    }
}

三个线程交替打印1-100（join）方法

public static  int num=0;
    public static void main(String[] args) {
        for(int i=0;i<3;i++){
            Thread thread1=new Thread(new theadrun(null),"A");
            Thread thread2=new Thread(new theadrun(thread1),"B");
            Thread thread3=new Thread(new theadrun(thread2),"C");
            thread1.start();
            thread2.start();
            thread3.start();
            try {
                Thread.sleep(100);
            } catch (InterruptedException e) {
                throw new RuntimeException(e);
            }
        }
    }
    public static class theadrun implements Runnable{
        private Thread thread;
        public theadrun(Thread thread){
            this.thread=thread;
        }
        @Override
        public void run() {
            if(num<=100){
                if(thread==null){
                    System.out.println(Thread.currentThread().getName()+num);
                }else{
                    try {
                        thread.join();
                    } catch (InterruptedException e) {
                        throw new RuntimeException(e);
                    }
                    System.out.println(Thread.currentThread().getName()+num);
                }
                num++;
            }
        }
    }

join方法在A线程中调用了B线程的join()方法时，表示只有当B线程执行完毕时，A线程才能继续执行。

三个线程交替打印1-100（Lock以及Lock+Condition方法）

private static Lock lock=new ReentrantLock();
    private static int num=0;
    public static void fun(int target){
        while(num<=10){
            lock.lock();
            if(num%3==target) {
                System.out.println(Thread.currentThread().getName() + num);
                num++;
            }
            //System.out.println("解锁");
            lock.unlock();
        }
    }
    public static void main(String[] args) {
        new Thread(()->{
            lockdemo1.fun(0);
        },"A").start();
        new Thread(()->{
            lockdemo1.fun(1);
        },"B").start();
        new Thread(()->{
            lockdemo1.fun(2);
        },"C").start();
    }

一样的思路，所有线程竞争锁，拿到锁的不符合条件也会解锁，符合条件的会输出并num++

Lock+Condition

private static int num=0;
    private static Lock lock=new ReentrantLock();

    private static Condition c1 = lock.newCondition();
    private static Condition c2 = lock.newCondition();
    private static Condition c3 = lock.newCondition();

    private static void  printABC(int targetNum, Condition currentThread, Condition nextThread) {
        while(num<=10) {
            lock.lock();
            try {
                while (num % 3 != targetNum) {
                    currentThread.await();  //阻塞当前线程
                }
                System.out.print(Thread.currentThread().getName());
                num++;
                nextThread.signal();    //唤醒下一个线程，而不是唤醒所有线程
            } catch (Exception e) {
                e.printStackTrace();
            }
            lock.unlock();
        }
    }
    public static void main(String[] args) {
        new Thread(()->{lockdemo2.printABC(0,c1,c2);},"A").start();
        new Thread(()->{lockdemo2.printABC(1,c2,c3);},"B").start();
        new Thread(()->{lockdemo2.printABC(2,c3,c1);},"C").start();
    }

与单纯的lock是一样的思路，但是通过condition实现对线程的精准唤醒，减少对同步锁的无意义竞争，浪费资源。
